# Embedded Sessions demo

This is a very simple plain HTML demo for embedding Examplary sessions using the [Embedded Sessions API](https://developers.examplary.ai/embed-sessions).
It shows how to create an embed session, load it in an iframe, and listen for events from the embed session using the `postMessage` API.
